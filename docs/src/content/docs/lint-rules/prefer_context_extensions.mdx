Prefer using context extensions to access a `Bloc` or `Repository` instance.

## Rationale

For consistency, prefer using context extensions like `context.read<T>()`,
`context.watch<T>()`, and `context.select<T, R>(...)` instead of
`BlocProvider.of<T>(context)` or `RepositoryProvider.of<T>(context)`.
